# 🎯 HabitQuest — Gamified Habit Tracker

**HabitQuest** — это не просто трекер привычек, это RPG-приключение для твоей продуктивности. Прокачивай своего персонажа, выполняя реальные задачи, зарабатывай XP и соревнуйся с друзьями в SDU!

---

## 🚀 Особенности
- **ИИ-Помощник:** Мультиязычный бот (KZ, RU, EN), который знает твой уровень и дает советы.
- **Геймификация:** Зарабатывай опыт (XP) за каждую привычку и повышай свой уровень.
- **Магазин (Shop):** Трать накопленные очки на кастомные стикеры и скины профиля.
- **Социалка:** Добавляй друзей и следи за их прогрессом в Лидерборде.
- **Аналитика:** Статистика и календарь твоих успехов.

---

## 🛠 Технологический стек
- **Frontend:** React, Tailwind CSS, i18next (Multilingual), React Router.
- **Backend:** Node.js, Express, Sequelize.
- **Database:** SQLite (легкая и быстрая).
- **AI:** Custom Rule-based Engine (Logic by Founders).
